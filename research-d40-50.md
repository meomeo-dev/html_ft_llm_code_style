C: 参数数量限制 - 0:无限制 / 1:建议不超过5个 / 2:最多4个 / 3:不超过3个超过重构为对象 / 4:严格最多2个 / 5:最多1个多参数通过数据类传入

E: 空行使用规范 - 0:随意添加无规则 / 1:函数之间至少一个空行 / 2:逻辑块之间一个空行分隔 / 3:明确规定每个结构周围空行数量 / 4:格式化工具强制统一 / 5:严格空行数量位置不允许偏差
F: 方法链格式化 - 0:链式调用可写一行 / 1:较长链式折行对齐点号 / 2:每个方法另起一行点号在行首 / 3:每个方法另起一行点号在行尾 / 4:强制拆分且点号位置统一 / 5:必须拆分并严格对齐
G: 尾随逗号 - 0:不使用 / 1:仅多行结构推荐 / 2:多行数组对象参数强制 / 3:所有多行列表含函数参数 / 4:单行结构也允许 / 5:所有可能地方都添加包括单行

I: 比较操作数顺序 - 0:变量常量位置随意 / 1:推荐常量在右侧 / 2:强制常量在右侧 / 3:使用Yoda条件 / 4:严格Yoda条件用于相等性 / 5:所有比较统一遵循Yoda或非Yoda静态检查

K: 类成员排序 - 0:无特定顺序 / 1:按类型分组但不严格 / 2:公共成员在私有之前 / 3:遵循特定顺序常量字段构造器方法 / 4:严格按预设规则排序 / 5:工具强制执行不得手动调整
